# Analytics API

Reference for the analytics helpers that compose core WFRMLS client endpoints into higher-level market and reporting workflows.

---

## Overview

`WFRMLSAnalytics` is not a raw RESO endpoint wrapper. It is a utility layer that uses the main client to calculate summaries, trends, and operational reports from existing resources.
